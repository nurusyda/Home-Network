Home Network (Beginner Level)

1. Network Diagrams

Logical Topology:

Below is the logical topology of the network:

  • Server → Router → Switch + Wireless Router → End Devices (PCs, Printer, etc.)
  • Wireless Router connected to the Router for Wi-Fi clients.

2. Configuration Guide

Step-by-Step Configuration:

Here’s an example of configuring the router:

  1. Configure Router LAN Interface: interface FastEthernet1/0
    ip address 192.168.0.1 255.255.255.0
    no shutdown
  2. Enable DHCP: ip dhcp pool HOME
    network 192.168.1.0 255.255.255.0
    default-router 192.168.1.1
    dns-server 8.8.8.8

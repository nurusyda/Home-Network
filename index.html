Home Network (Beginner Level)

1. Network Diagrams

Logical Topology:

  • Server → Router → Switch + Wireless Router → End Devices (PCs, Printer, etc.)
  • Wireless Router connected to the Router for Wi-Fi clients.

IP Addressing Scheme:

  • Server IP: 8.8.8.8/24
  • Default Gateway: 192.168.0.1/24
  • DHCP Range: 192.168.0.0/24

2. Configuration Guide

Step-by-Step Configuration:

  1. Configure Router LAN Interface: interface FastEthernet1/0
    ip address 192.168.0.1 255.255.255.0
    no shutdown
  2. Enable DHCP: ip dhcp pool HOME
    network 192.168.1.0 255.255.255.0
    default-router 192.168.1.1
    dns-server 8.8.8.8
  3. Connect Wireless Router to Main Router.
  4. Set WPA2 wireless security.

Device Configurations:

  • Router: Basic setup with DHCP.
  • Wireless Router: Default gateway 192.168.1.1, DHCP enabled.

Security Implementations:

  • Set up WPA2 encryption on the Wireless Router.

Troubleshooting Procedures:

  • Check IP address allocation from DHCP.
  • Verify connectivity using ping from wired and wireless devices.

3. Test Plans

Connectivity Tests:

  • Ping Router from PC (e.g., ping 192.168.1.1).
  • Ping a public DNS server (e.g., ping 8.8.8.8) to confirm internet access.
